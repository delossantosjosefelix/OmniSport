## Project Objective

The objective of this project is to develop a mobile application that provides basic information about different sports while applying the concepts learned in the Mobile Device Programming course. The application serves as an educational and informational platform that allows users to explore various sports and learn about some of their most notable athletes.

## Description

OmniSport is a mobile application developed in Android Studio that provides introductory information about four popular sports: basketball, soccer, baseball, and boxing. Each section contains a brief overview of the sport along with information about selected athletes who have made significant contributions to their respective disciplines.

## Included Sports

* Basketball
* Soccer
* Baseball
* Boxing

## Features

* Simple and user-friendly interface
* Sports categorized by discipline
* Brief description of each sport
* Information about notable athletes
* Easy navigation between sections

## Technologies Used

* Android Studio
